StockAI
Análise Preditiva de Ações

Dashboard de análise financeira com machine learning para predição de tendências em ações da B3 e NYSE. Utiliza modelos LSTM e Prophet para forecasting de séries temporais, visualizações interativas com Plotly e consumo da API yfinance para dados históricos em tempo real.

📊 Dashboard Principal
📈 Gráficos de Predição
🤖 Modelo ML
🔔 Alertas de Preço

Minha Participação

  • Pipeline de coleta e limpeza de dados financeiros com Pandas e yfinance
  • Implementação de modelo LSTM com TensorFlow/Keras para forecasting de preços
  • Comparação e avaliação de modelos (LSTM vs Prophet vs ARIMA) com métricas MAE/RMSE
  • Dashboard interativo com Plotly Dash e gráficos candlestick atualizados ao vivo
  • Sistema de alertas por e-mail quando preço cruza thresholds definidos pelo usuário
  • Análise de sentimento de notícias financeiras com BERT para enriquecer predições

Tecnologias

Python TensorFlow Scikit-learn Pandas Plotly Dash Prophet yfinance BERT Jupyter
