ECOmida

Projeto desenvolvido com React no frontend e Flask + MongoDB no backend. O ECOmida permite apoiar e aprimorar o trabalho de combate à fome por meio de doações e reaproveitamento de alimentos, conectando doadores a organizações que precisam, com uma interface intuitiva e acessível para todos os perfis de usuário.

Minha Participação

  • Elaborei o backend completo da aplicação com Python e Flask (API RESTful)
  • Modelei e implementei o banco de dados não relacional com MongoDB
  • Desenvolvi os endpoints de doações, reaproveitamento e gestão de usuários
  • Integrei o backend com o frontend React via consumo de API REST
  • Realizei o deploy do frontend na Vercel e configurei o ambiente de produção

Tecnologias

React.js CSS3 Node.js Python Flask MongoDB Git GitHub
