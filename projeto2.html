VitaMed

Sistema web para agendamento de consultas e gestão básica de usuários, com autenticação, área de perfil e fluxo de recuperação de senha por e-mail. O VitaMed facilita o agendamento de consultas e exames médicos com especialistas, entregando uma experiência fluida e segura para pacientes e clínicas.

Minha Participação

  • Participei ativamente no desenvolvimento de todo o backend com Node.js e Express
  • Modelei o banco de dados relacional (MER/DER) e implementei com Sequelize e MySQL
  • Configurei autenticação com Passport.js e sessões seguras com Express Session
  • Implementei o fluxo de recuperação de senha por e-mail via Nodemailer e Mailtrap
  • Realizei o deploy da aplicação na Vercel com variáveis de ambiente configuradas

Tecnologias

Node.js Express Handlebars Sequelize MySQL Passport.js Nodemailer Bootstrap Git GitHub
