Full Stack Web App

TaskFlow
Gerenciador de Tarefas

Aplicação web full-stack para gestão produtiva de tarefas e projetos. Desenvolvida com React no front-end e Node.js + Express no back-end, com banco de dados MongoDB. Inclui autenticação JWT, interface drag-and-drop, notificações em tempo real via WebSocket e dashboard de produtividade com gráficos.

🗂️ Dashboard Principal
📋 Lista de Tarefas
📊 Relatórios
👤 Perfil do Usuário

Minha Participação

  • Arquitetura completa do sistema, desde o banco de dados até a interface
  • Implementação do sistema de autenticação com JWT e refresh tokens seguros
  • Desenvolvimento da funcionalidade de drag-and-drop com React DnD
  • Integração de WebSockets para notificações em tempo real entre usuários
  • Deploy na AWS EC2 com Nginx como proxy reverso e certificado SSL
  • Testes unitários e de integração com Jest e Supertest (cobertura >80%)

Tecnologias

React Node.js Express MongoDB JWT WebSocket Tailwind CSS Jest Docker AWS EC2
