Academia

Estamos desenvolvendo um site para uma academia, utilizando as tecnologias HTML, CSS e JavaScript. Nosso objetivo é criar um site totalmente responsivo e funcional para atender às necessidades da academia e seus clientes. A plataforma oferece informações completas sobre os serviços, horários de funcionamento, instrutores, planos de assinatura e muito mais.

Minha Participação

  • Auxiliei na elaboração do protótipo e wireframes para definição do layout das páginas
  • Participei da criação da documentação técnica do projeto
  • Realizei o deploy do site via GitHub Pages, configurando o repositório remoto
  • Colaborei com a equipe no versionamento do código usando Git e GitHub
